# SECTION 4: CRYPTOGRAPHIC AUDITABILITY

## 4.1 Proof Certificate Architecture

Each recommendation generates a cryptographic audit certificate containing:
- Proof Hash: SHA-256 hash of recommendation data
- Timestamp: Server-generated, immutable
- Asset/Thesis: Plain English reasoning
- Confidence Score: Quantitative assessment
- Status: Cleared or Risk-Rejected

## 4.2 Merkle Chain

Certificates are linked in a merkle chain providing tamper evidence.
